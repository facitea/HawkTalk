# HawkTalk

## 1차 기획안(학습 목표)
> ### 개요
- Social Media/Messenger 융합
- 차세대 메신저 서비스 구축

> ### 목적
- 1020세대를 타겟으로 오락성을 갖춘 가벼운 웹애플리케이션 개발

> ### 기술
- 웹 애플리케이션 기반(Javascript)
- node.js / express.js 기반 서버
- React.js

> ### 기능/기술
- Social Media로서 간단한 CRUD 기능 구현(twitter 벤치마킹)
- 간단한 

> ### 수익모델
- 광고연동

> ### 부가기능(오락요소 등/금지어 방장설정가능 등)
